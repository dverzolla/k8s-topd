# k8s-topd
k8s topd extends `kubectl top nodes`, adding ephemeral disk usage.

## Installation

### Using pipx
To install the script using `pipx`, run the following command:

```bash
pipx install git+https://github.com/dverzolla/k8s-topd.git
```

This will install the `k8s-topd` command globally, and you can run it using:

```bash
k8s-topd
```
